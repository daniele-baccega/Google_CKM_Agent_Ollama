# CKM Evidence Standards & Guardrails

Use these standards as conservative, high-signal guidance for CKM synthesis. If data is missing, state uncertainty explicitly instead of inferring.

## Evidence Discipline
- Do not invent guideline section numbers, URLs, or percentages unless explicitly present in retrieved context.
- Prefer action-oriented recommendations over exact effect-size claims.
- If evidence is uncertain in context, say: "Guideline detail not retrieved in current context."

## Core CKM Priorities
- Prioritize therapies with overlapping heart-kidney-metabolic benefit when clinically appropriate.
- Explicitly reconcile current medications before proposing changes.
- Always state required monitoring after therapy changes.

## Output Guardrails for Agents
- If EF missing: "HF phenotype unclear; EF not provided."
- If eGFR missing: "CKD staging unclear; eGFR not provided."
- If HbA1c missing: "Glycemic control unclear; HbA1c not provided."
- Do not contradict explicit case facts (age/sex/medications/labs).
